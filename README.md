# Dinner-Picker-App
A small app I created that helps you decide if you are eating fast food, dining in somewhere, or staying home and eating something there. Each category is an Array that has different restaurants, or for eating at home, different dinner ideas for you to eat.


## MAIN FUNCTIONALITIES
-Randomly selects a restaurant or dinner idea from multiple categories
-Displays said result in a JavaFX window
-Beginner-friendly design

## TECHNOLOGIES USED
-JAVAFX, JAVA, Eclipse IDE.

## HOW TO RUN/INSTALL
1.) Clone the repo
2.) Open in Eclipse
3.) Run Main.java
